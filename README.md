# gtbs2-api
This is the API that hosts metadata and images for stage 2 of the Grand Theft Bacon game.

The API hosts images and metadata for the cars contract and items contract.

# Setup Instructions
1. Clone this repository
2. Run the following command to install all dependencies
```
npm install
```
3. Run the development application
```
npm run dev
```

# Other info
The development server is set to port 80 by default
